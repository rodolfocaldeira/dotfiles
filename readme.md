# My (Rodolfo's) dotfiles

## Install

To install simply do:

- `git clone git@bitbucket.org:rodolfocaldeira/dotfiles.git`
- `cd dotfiles`
- `sh install.sh` 

The install script will symlink the files to your home directory.
